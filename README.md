# Análise de Dados de Cancelamento de Clientes
Projeto de análise de dados desenvolvido em Python com o objetivo de identificar padrões e fatores associados ao cancelamento de clientes (churn), contribuindo para tomadas de decisão mais estratégicas.
A análise busca entender os principais motivos que levam clientes a cancelarem um serviço, permitindo a criação de estratégias de retenção.

## Objetivos
* Analisar dados de clientes;
* Identificar padrões de cancelamento;
* Gerar insights para retenção de clientes;
* Aplicar técnicas de análise de dados com Python.

## Etapas da análise
O projeto segue um fluxo típico de análise de dados:
1. Importação e exploração dos dados;
2. Tratamento e limpeza da base;
3. Análise exploratória (EDA);
4. Identificação de padrões e variáveis relevantes;
5. Geração de insights.

## Tecnologias utilizadas
* Python;
* Pandas;
* Matplotlib / Seaborn;
* Jupyter Notebook.

## Principais insights
A análise exploratória revelou padrões relevantes associados ao cancelamento de clientes:
* Clientes com **contrato mensal** apresentam taxa de cancelamento significativamente maior;
* Clientes que realizaram **mais de 4 ligações ao call center** tendem a cancelar o serviço;
* Atrasos superiores a **20 dias no pagamento** estão fortemente associados ao cancelamento.

Esses resultados indicam possíveis problemas relacionados à experiência do cliente, modelo de contrato e inadimplência.


## Interpretação de negócio
* Contratos mensais podem indicar menor fidelização;
* Alto volume de chamadas ao suporte pode refletir insatisfação;
* Atrasos no pagamento podem ser um forte indicador de risco de cancelamento.

## Possíveis melhorias
